树莓派4安装安卓9系统

我在8月份左右买了个树莓派4，玩了几天以后就一直吃灰了。直到某一天，我的小米盒子3增报废了，于是他就担任起了电视盒子的作用

起初我是准备装Android系统的，可惜那个时候没有为树莓派4开发的安卓系统，所以我只能安装了Manjaro Linux，这个Linux安装了KDE桌面，非常华丽，可是有时候会有点卡顿，但是我忍了（其实是找不到其他好看又好用的系统了

今天，我Google的时候，居然找到了树莓派4的安卓9系统，我就搬运过来和大家分享。

1.下载链接

由于原来的下载速度比较缓慢，我爬梯/子才下载下来，所以我提供一个百度网盘链接和我自己下载站链接

本地下载:网页链接
百度网盘下载:网页链接 提取码: qh8u

如果链接失效请联系博主

2.烧录固件

请参考网页链接，道理都一样的，都是把固件烧录到TF卡，然后用DG扩容

3.关于按键

这个我不确定是否对所有键盘都有效，但是我键盘上F1键返回桌面，F2是返回键，F4是任务，F5是电源键，F11音量-，F12音量+

电源键需要鼠标手动点击，不知道为什么

作者给出的物理电源开关接线

4.UI预览

5.一些问题

1.分辨率最高只支持720P

在原作者的页面上，写明最高支持720P，我也尝试过1080P和1080Pi，也全部无法显示。

Q: My display is not working. I can only see the rainbow screen but no Android boot animation. What should I do?

A: This build uses 1280×720 resolution by default so you need to use a HDMI display that supports it. See this page under ‘Which values are valid for my monitor?’ to see how to check which resolutions your display supports using Raspbian. If your display doesn’t support 1280×720 resolution, you can try changing values in /boot/config.txt to something it does (see this page). 720p is still the maximum resolution that is supported by the graphics drivers used in this build.

作者原文

附上原文链接:网页链接

2.WIFI连接提示’此网络无法访问互联网’

这个应该是安卓原生系统的通病了，大家用adb命令修复一下就好了

另外如果有线连接部分软件提示’未连接互联网’，请试试无线连接
